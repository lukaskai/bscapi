# Binance Smart Chain Data API

****
Supported protocols:
* ApeSwap
* PancakeSwap
* Beefy
* AutoFarm
* Venus
****

**Setup**

1. Modify .env.mainnet and setup BSC node endpoint, decimals, port, etc.
2. Run `npm install`

**Development**

1. Run `npm run start`

**Production**

1. Run `npm run start-prod`
